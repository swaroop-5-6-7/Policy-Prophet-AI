Architectural Ledger API

Complete technical documentation, API references, and integration guides for the Policy Prophet AI prediction engine.

Overview

Policy Prophet AI is a high-fidelity, end-to-end Machine Learning web application designed to predict health insurance coverage costs with institutional precision.

The project combines a robust Python-based Random Forest Machine Learning model with a premium, corporate-style web interface ("Architectural Ledger") to provide users with real-time actuarial risk assessments.

Architecture

The application architecture separates the presentation layer from the predictive logic.

  • Frontend: A static, responsive interface built with HTML5, Vanilla CSS, and Vanilla JavaScript. It captures user inputs and visualizes predictions dynamically using Chart.js.
  • Backend: A Python Flask API that loads the pre-trained Machine Learning model (`model.pkl`) into memory and exposes RESTful endpoints for real-time predictions.
  • Model: A robust Scikit-Learn `RandomForestRegressor` trained on historical Medical Cost Personal Datasets. Variables include Age, Biological Sex, BMI, Dependents, Tobacco Usage, and Geographic Region.

API Reference

The prediction engine exposes a single REST endpoint for calculating estimated premium costs.

Predict Premium

POST /predict

Calculates the annual health insurance premium based on demographic and health indicators.

Request Payload (JSON)

{ "age": 35, "sex": "Male", "bmi": 28.5, "children": 1, "smoker": "No", "region": "Northeast" }
  • age (integer): The age of the applicant (0-150).
  • sex (string): "Male" or "Female".
  • bmi (float): Body Mass Index (10.0 - 60.0).
  • children (integer): Number of dependents (0-5+).
  • smoker (string): Tobacco usage ("Yes" or "No").
  • region (string): Geographic location ("Northeast", "Northwest", "Southeast", "Southwest").

Success Response

{ "predicted_cost": 4725.32 }

Local Development

To run the application locally, ensure you have Python 3 installed.

1. Install Dependencies

pip install flask flask-cors pandas numpy scikit-learn

2. Start the Backend API

python app.py

The server will start running on http://127.0.0.1:5000.

3. Launch the Frontend

Open index.html in your preferred browser and start making predictions.

Optional: If you wish to retrain the model, run python train_model.py before starting the Flask server.
